## 배운 것

- `content/til`, `content/blog` 같은 폴더 이름 = 분류 기준(트랙)
- 새 글 = 해당 폴더에 `.mdx` 하나 넣으면 끝
- 목록·라우팅은 폴더를 읽어 자동 생성

## 왜 좋았나

- 태그 목록·카테고리 테이블 같은 별도 설정 불필요
- "폴더 = 트랙" 규칙 하나로 구조 단순해짐

## 다음

- 트랙 추가 = 폴더만 추가하면 되는지 확인
- 라우팅 검증 위치 확인
