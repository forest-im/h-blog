# 연속과 이산

![연속,이산그래프](https://imagedelivery.net/5LG7d9X3kuXKRQ06TLYDSg/c1d42c97-745a-463d-5cd4-af89fbcf3800/public)

## 연속(continuous)

중단 또는 급격한 변화가 없는 상태

- 연속수학: 이러한 연속적인 수학구조에 대해 연구하는 수학을 연속수학이라고 한다. (미적분학, 위상수학, 복소수론, 해석학 등)
  - 연속수학에서는 실수와 같이 연속적인 특징을 가지는 집합에 관심을 가지고, 이들은 '셀 수 없다'는 특징을 가지고 있다. 그래프로 표현하면 "선"의 형태로 표현된다.

## 이산(discrete)

떨어져 있는, 불연속적인 상태

- 이산수학: 연속적이지 않고, 뚜렷하게 구분되는 수학적 구조를 다루는 수학 분야.
- 일반적으로 디지털 컴퓨터가 다루는 데이터는 0과 1처럼 이산적인 값을 가지므로, 이산수학은 디지털 컴퓨터가 데이터를 처리하는 과정에 필요한 수학적인 개념을 제공해 줌

### 이산적인 데이터 집합

- 이산적인 데이터 집합은 셀 수 있는 개별 항목들로 구성된 데이터를 말한다.
- 전부 정수 값으로 표현할 수 있다.
- 예) 가존 구성원 수, 하루동안 은행에 들어온 고객 수, 주식의 가격 등

# 모델링과 추상화

## 도구, 기법, 방법론

- 문제 해결을 위해 도구, 기법, 방법론 등이 필요함
- 수학에는 많은 도구, 기법, 방법론이 존재함
  - 수학의 정의와 정리들은 모두 수학적 도구로 볼 수 있다.

| 도구              | 기법              | 방법론       |
| ----------------- | ----------------- | ------------ |
| 수학의 정의, 정리 | 가감법, 대입법 등 | 근의 공식 등 |

## 수학적 모델링

- 현실 세계의 문제나 현상을 **수학적 구조(방정식, 함수, 확률 등)**로 표현하는 것.
- 수학적인 도구, 기법, 방법론을 이용해 문제를 해결하기 위해서는 먼저 문제를 수학적 모델로 변화시켜야 한다.

## 데이터 모델링

- 현실 세계의 정보를 **데이터 구조(테이블, 속성, 관계 등)**로 표현하는 것
  → 예: 고객-주문 관계를 ER 다이어그램으로 표현

## 추상화

- 문제 해결을 위해 필요한 핵심 내용만 남기고 관련이 없는 내용을 제거함으로써 문제를 단순화시키는 과정
